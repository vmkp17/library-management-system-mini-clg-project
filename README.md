# 📚 Library Management System (College Mini Project)

A beginner-friendly **Library Management System** developed using **Python, Flask, SQLite, HTML, CSS, and Bootstrap** as part of my **B.Tech College Mini Project**.
The main objective of this project was to understand #how Python applications interact with databases, implement a simple web interface, and perform basic data retrieval and search operations.

## 📖 Project Overview
This application allows users to browse books by category, search for books, and view library statistics through a clean web interface.
The project focuses on:
- Database design
- Data storage and retrieval
- Backend development using Flask
- Basic frontend development
- Search functionality
- Simple project organization


## ✨ Features
- 📚 Dashboard with library overview
- 📂 Browse 10 different book categories
- 📖 View books by category
- 🔍 Search books by:
  - Book ID
  - Book Name
  - Author
  - Publisher
- 📊 Library statistics
- 💾 SQLite database integration
- 🌐 Responsive web interface
